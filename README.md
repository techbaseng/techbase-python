## About This Course

Python is one of the world's most popular and beginner-friendly programming languages. It is used in data science, web development, automation, artificial intelligence, and much more.

In this course, you will go from writing your very first line of code all the way to building real programs that solve everyday problems — with examples and exercises set right here in Nigeria.

**No prior programming experience needed. Just curiosity and a computer.**

---

## What You Will Learn

By the end of this course you will be able to:

- Write, run, and debug Python programs
- Work with variables, data types, and operators
- Take user input and display formatted output
- Use conditions and loops to control program flow
- Define and call your own functions
- Work with lists, tuples, and dictionaries
- Read from and write to text files
- Build a complete, practical Python project from scratch
